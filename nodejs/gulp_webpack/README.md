## Setup enviroment

Setup nodejs from (nodejs)[https://nodejs.org/en/download/]
`npm` has been integrated with nodejs.

Setup gulp and webpack
```
npm install webpack -g
npm install webpack-cli -g
npm install gulp -g

```
## Build target
Run the following steps
```
gulp run
```
